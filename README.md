# Proyecto 3er Trimestre – UD7. Persistencia de objetos

## Descripción
Aplicación en Java para la gestión de proyectos y desarrolladores usando persistencia de objetos y patrón DAO.

## Funcionalidades
- CRUD de proyectos y desarrolladores
- Asignación y eliminación de relaciones
- Consultas sobre proyectos y desarrolladores
- Ejecución de pruebas desde `Main`

## Tecnologías
- Java
- DAO
- Persistencia de objetos

## Nota
El `Main` es reutilizable y puede ejecutarse varias veces sin alterar los resultados.
